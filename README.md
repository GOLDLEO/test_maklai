# test_maklai

## ON LINUX 
```
$ mkdir test_maklay (or your name directory)
$ cd test_maklay
$ git clone https://github.com/GOLDLEO/test_maklai.git
$ virtualenv env 
$ source env/bin/activate
(env)$ cd test_maklay
(env)$ pip install -r requirments.txt
(env)$ python3 manage.py migrate
(env)$ python3 manage.py createsuperuser
(env)$ python3 manage.py runserver
```
